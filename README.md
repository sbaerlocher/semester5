# Installation

Docker und Docker-compose muss installiert sein

```bash
docker-compose build
```

```bash
docker-compose up -d
```

Datenbank installation

```bash
 docker exec -it semester5_app_1 bash
```

```bash
compose install
```

```bash
php artisan migrate
```
